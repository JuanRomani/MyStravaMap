# MyStravaMap

An one-weekend personal project that displays all my Strava exercises in one single map.

## Libraries / Frameworks used

- [Spring Boot v2.0.3, by Pivotal](https://spring.io/projects/spring-boot)
- [AngularJS v1.4.3, by Google](https://angularjs.org/)
- [javastravav3api v2.0.0-alpha, by danshannon](https://github.com/danshannon/javastravav3api)
- [Leaflet v1.3.1, by Vladimir Agafonkin](https://leafletjs.com/)
- [Leaflet.encoded v0.0.8, by jieter](https://github.com/jieter/Leaflet.encoded)
